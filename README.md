FutGestorPro

Aplicativo desenvolvido em React Native com Expo para gerenciamento de elenco de futebol.
Projeto da disciplina Programação para Dispositivos Móveis - ADS 4° Período.


Como executar o projeto:

Clone este repositório: git clone https://github.com/Bentim-Luccas/av2-prog-para-disp-moveis-futgestorpro.git

Acesse a pasta do projeto: cd FutGestorPro-Luccas

Instale as dependências: npm install

Inicie o projeto: npx expo start

Escaneie o QR Code com o aplicativo Expo Go (Android/iOS) ou abra no emulador configurado.


Funcionalidades:

Exibe status do elenco: Treinamento ou Partida Oficial.
Botão para alternar status com mudança de cor de fundo e texto.
Alerta inicial confirmando sincronização dos dados do plantel.
Lista de jogadores com navegação para tela de detalhes.
Tela de detalhes mostrando: Nome, posição e número de gols


Tecnologias utilizadas:

React Native,
Expo,
React Navigation
